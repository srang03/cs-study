# 로드맵

## 1. 시니어의 가치가 드러나는 순간

실력 있는 시니어의 존재가 빛나는 순간은 장애 대응이다.

- 장애 상황 인지 그 자체도 능력이다.
- 빠른 대응과 근본원인 규명은 별개의 문제이다.
- 어떤 경우라도 인프라에 대한 높은 이해도는 필수이다.

## 2. 본인의 역할

- 기본에 충실할 것
  - 프로그래밍 언어
  - 자료구조 알고리즘
    - 코딩 테스트 준비
  - 운영체제
    - 윈도우
    - 리눅스
  - 컴퓨터구조
  - 데이터베이스
    - RDBMS
    - NOSQL (Document, Key-value, Column, Graph)
  - 컴파일러
- 문제에 대해 함께 의견을 교환할 수 있어야 한다.
