# 1.1 환경과 실행

> 랭크: Bronze (1/5)

---

## 1. 설명

### 자바 프로그램의 실행 흐름

자바 프로그램은 소스 코드 → 컴파일 → 바이트코드 → JVM 실행 순서로 동작한다.

```
HelloWorld.java  →  javac  →  HelloWorld.class  →  java  →  실행
   (소스)          (컴파일)      (바이트코드)        (JVM)
```

### javac (컴파일러)

`.java` 파일을 `.class` 파일(바이트코드)로 변환한다.

```bash
javac HelloWorld.java
```

- 문법 오류가 있으면 컴파일 에러 발생
- 성공하면 같은 디렉터리에 `HelloWorld.class` 생성

### java (JVM 실행)

`.class` 파일의 바이트코드를 JVM이 실행한다.

```bash
java HelloWorld
```

- `.class` 확장자를 붙이지 않는다
- JVM이 해당 클래스의 `main` 메서드를 찾아 실행한다

### main 메서드

모든 자바 프로그램의 시작점이다.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

- `public`: JVM이 외부에서 접근 가능해야 한다
- `static`: 객체 생성 없이 실행 가능해야 한다
- `void`: 반환값이 없다
- `String[] args`: 명령줄 인수를 배열로 받는다

### 파일 규칙

- 파일명 = public 클래스명: `HelloWorld.java` 안에 `public class HelloWorld`
- 하나의 `.java` 파일에 `public` 클래스는 하나만 허용

### 빌드 타임과 런타임

| 시점 | 수행 주체 | 하는 일 |
|------|-----------|---------|
| 빌드 타임 | javac | `.java` → `.class` 컴파일 |
| 런타임 | JVM | ClassLoader가 `.class`를 로딩하고 실행 |

자바에는 별도 링크 단계가 없다. C/C++에서 링커가 하는 클래스 해석(resolution) 작업은 JVM이 런타임에 수행한다.

## 2. 예제

### 예제 1: Hello World

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

```bash
$ javac HelloWorld.java
$ java HelloWorld
Hello, World!
```

### 예제 2: 명령줄 인수 사용

```java
public class Greeting {
    public static void main(String[] args) {
        // args[0]에 첫 번째 명령줄 인수가 들어온다
        System.out.println("Hello, " + args[0] + "!");
    }
}
```

```bash
$ javac Greeting.java
$ java Greeting Java
Hello, Java!
```

### 예제 3: 여러 줄 출력

```java
public class MultiLine {
    public static void main(String[] args) {
        System.out.println("첫 번째 줄");  // 줄바꿈 포함
        System.out.print("두 번째 줄");     // 줄바꿈 없음
        System.out.print(" 이어서 출력");
        System.out.println();               // 줄바꿈만
        System.out.println("세 번째 줄");
    }
}
```

```
첫 번째 줄
두 번째 줄 이어서 출력
세 번째 줄
```

## 4. 이론 퀴즈

Q1. `javac` 명령은 무엇을 하는가?

<details>
<summary>정답</summary>
.java 소스 파일을 .class 바이트코드 파일로 컴파일한다.
</details>

Q2. `java HelloWorld.class`로 실행하면 어떻게 되는가?

<details>
<summary>정답</summary>
오류 발생. java 명령에는 .class 확장자를 붙이지 않는다. `java HelloWorld`가 올바른 실행 방법이다.
</details>

Q3. `main` 메서드가 `static`이어야 하는 이유는?

<details>
<summary>정답</summary>
JVM이 프로그램 시작 시 객체를 생성하지 않고 바로 메서드를 호출해야 하므로, 인스턴스 없이 호출 가능한 static이어야 한다.
</details>

Q4. 자바에서 링크(클래스 해석)는 언제 일어나는가?

<details>
<summary>정답</summary>
런타임에 JVM이 수행한다. 자바에는 별도 링크 단계가 없다.
</details>

Q5. 파일명이 `Test.java`인데 안에 `public class Hello`를 선언하면?

<details>
<summary>정답</summary>
컴파일 에러. public 클래스명과 파일명이 일치해야 한다. `Hello.java`로 저장하거나 클래스명을 `Test`로 바꿔야 한다.
</details>

## 5. 함정 노트

### 1. `.class` 확장자 붙이기

```bash
# 틀림
java HelloWorld.class

# 맞음
java HelloWorld
```

### 2. 파일명과 클래스명 불일치

`public class`의 이름은 반드시 파일명과 같아야 한다. 백준에서는 `Main`, 프로그래머스에서는 `Solution`이 고정 클래스명이다.

### 3. `println` vs `print`

- `println`: 출력 후 줄바꿈
- `print`: 출력 후 줄바꿈 없음

출력 형식이 틀려서 오답 처리되는 경우가 있다.

### 4. `main` 시그니처 오타

```java
// 틀림 - 실행 안 됨
public static void main(String args) { }    // 배열이 아님
public static void Main(String[] args) { }  // 대문자 M

// 맞음
public static void main(String[] args) { }
```

## 6. 코딩 문제

### 문제 1: Hello World

난이도: Bronze V

문제  
"Hello, World!"를 정확히 한 줄로 출력하는 프로그램을 작성하라.

입력  
없음

출력  
```
Hello, World!
```

핵심: `System.out.println` 사용, main 메서드 시그니처 정확히 작성  
힌트: 한 줄 출력이 전부. `public static void main(String[] args)` 구조를 정확히 작성하는 것이 포인트

---

### 문제 2: 두 줄 출력

난이도: Bronze V

문제  
동일한 문구를 두 줄에 걸쳐 출력하라.

입력  
없음

출력  
```
강한 친구 대한육군
강한 친구 대한육군
```

핵심: 동일한 문자열을 두 줄 출력  
힌트: `println`을 두 번 호출하거나, 같은 문자열을 두 번 출력

---

### 문제 3: 고양이

난이도: Bronze V

문제  
아래와 같이 고양이 모양을 출력하라. 역슬래시(`\`)가 포함된다.

입력  
없음

출력  
```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

핵심: 특수문자(`\`)가 포함된 출력  
힌트: 역슬래시를 출력하려면 `\\`로 이스케이프. `println`으로 각 줄 출력
