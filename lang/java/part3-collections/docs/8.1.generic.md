# Java 제너릭(Generic) 학습

## 1. 제너릭의 개념과 목적

Generic은 사전적으로 "일반적인"이라는 뜻이다. Java에서 제너릭은 추상 자료형(ADT, Abstract Data Type)을 표현하는 방식으로, 특정 구체 타입에 대한 의존을 줄이고 범용 코드를 만들어 재사용성을 높인다. C++의 템플릿과 개념적으로 유사하다.

- 특정 형식에 대한 의존성 감소
- 형 안정성 유지와 코드 재사용 극대화
- 런타임 타입 캐스팅 없이 컴파일 타임에 타입 검사

실제로 List, Set 등 자료구조는 보통 제너릭 클래스로 구현되어 있으며, 이들 API는 Java Collection Framework로 제공된다.

---

## 2. 제너릭 타입의 기본 사용

일반 클래스에 타입 파라미터를 두면 제너릭 클래스가 된다.

```java
class ClassName<TypeParameter> { }
```

예: `MyData<T>`처럼 `<T>`를 붙이면, 그 클래스 안에서 `T`는 "아직 정해지지 않은 타입"을 나타낸다. 사용할 때 `MyData<String>`, `MyData<Integer>`처럼 구체 타입을 지정한다.

### 2.1 타입 매개 변수 이름 관례

타입 매개 변수는 보통 한 글자로 쓰며, 다음 관례를 자주 사용한다.

| 문자 | 의미 |
|------|------|
| E | Element (컬렉션 원소) |
| K | Key (맵의 키) |
| N | Number (숫자) |
| T | Type (일반 타입) |
| V | Value (맵의 값) |

- 컴파일 시점에 타입 변수는 지정한 실제 타입으로 치환되어 처리된다(타입 소거).
- 비제너릭 코드(예: `Object`를 담는 컨테이너)는 값을 꺼낼 때마다 형변환이 필요하고, 잘못된 타입으로 캐스팅하면 런타임 오류가 난다. 제너릭을 쓰면 컴파일 시점에 타입이 맞는지 검사할 수 있다.

참고: JVM에는 제너릭 정보가 소거된 바이트코드만 남기 때문에, 내부적으로는 필요 시 타입 변환 코드가 생성될 수 있다. 다만 개발자가 직접 캐스팅을 쓰지 않아도 되고, 컴파일러가 타입 안전성을 보장해 준다.

## 3. 타입 제한과 제약사항

### 3.1 타입 상한 (extends)

타입 변수를 특정 클래스나 그 하위 타입으로만 쓰고 싶을 때 상한을 둘 수 있다.

```java
<T extends Parent>
```

이렇게 하면 `T`는 반드시 `Parent` 또는 그 서브클래스여야 한다.

### 3.2 제약사항

- 타입 변수 자리에는 클래스(참조 타입)만 올 수 있다. `int`, `double` 같은 원시형은 사용할 수 없으며, 필요한 경우 `Integer`, `Double` 같은 래퍼 타입을 사용한다.
- 제너릭 클래스 내부에서 static 멤버(필드·메서드)는 해당 클래스의 타입 파라미터 `T`를 직접 사용할 수 없다. `T`는 `MyData<String>`, `MyData<Integer>`처럼 구체 타입이 정해질 때 비로소 의미가 정해지는데, static 멤버는 인스턴스가 아닌 클래스에 하나만 존재한다. 따라서 "하나뿐인" static 멤버가 `T`가 String일 때와 Integer일 때를 동시에 만족할 수 없어, 컴파일러는 static 문맥에서 `T`를 허용하지 않는다
- 인스턴스(또는 파라미터화된 타입)에 대한 제너릭이지, 클래스 전역이 아니라서, 클래스 전역인 static 멤버에는 제너릭 T를 쓸 수 없다.

---

## 4. 제너릭 메서드

클래스 전체가 아닌 특정 메서드에만 제너릭을 적용할 수 있다. 이때 타입 파라미터는 해당 메서드에 한정되므로, 클래스의 `T`와 달리 static 메서드에서도 자신만의 타입 파라미터를 선언해 사용할 수 있다.

```java
public static <T> T testMethod(T param) {
    return param;
}
```

메서드 반환 타입 앞의 `<T>`가 메서드 타입 파라미터 선언이며, 호출 시 전달되는 인자나 명시적 타입으로 `T`가 추론된다.
