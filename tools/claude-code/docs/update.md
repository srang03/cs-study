# Claude Code 업데이트 요약

## 기능 개요

| 기능 | 명령어 | 목적 | 핵심 동작 | 적합한 상황 |
|------|--------|------|-----------|-------------|
| Simplify | /simplify | 코드 자동 정리 | 3개 리뷰 에이전트 병렬 실행 (재사용·품질·효율성 검토) | 기능 구현/버그 수정 후 정리 |
| Batch | /batch | 대규모 병렬 변경 | 조사 → 5~30개 단위 분해 → 승인 → git worktree 병렬 실행 → 구현·테스트·PR 자동 | 코드베이스 전체 대규모 변경 |
| Remote Control | /remote-control | 원격 로컬 환경 사용 | 파일시스템, MCP, 도구, 프로젝트 구성 모두 원격 사용 가능 | 원격 개발 환경 접근 |
| Ralph Loop | /ralph-loop "프롬프트" --max-iterations N --completion-promise "DONE" | 자동 반복 피드백 루프 | 프롬프트 실행 → 작업 수행 → 종료 시도 → 완료 조건 미만족 시 재실행 | 명확한 성공 기준과 반복 개선이 필요한 작업, 그린필드·TDD |
