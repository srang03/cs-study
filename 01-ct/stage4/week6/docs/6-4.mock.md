<details>
<summary>정답</summary>

```java
int[][] copy = new int[n][m];
for (int i = 0; i < n; i++)
    copy[i] = grid[i].clone();
```

## 1. 설명

### 실전 모의고사의 목적

지금까지 배운 모든 내용을 **시간 제한** 안에 종합 활용하는 연습이다.

- stage1~3: 문법, 자료구조
- week1~3: 복잡도, 정렬, 스택/큐, 해시
- week4: 트리, 힙, 그래프, DFS, BFS
- week5: DP
- week6 1~3: 기출 유형

### 시간 제한 모의고사 가이드

| 항목 | 카카오 모의 | 삼성 모의 | LINE 모의 |
|------|-----------|----------|----------|
| 시간 | 5시간 | 3시간 | 2.5시간 |
| 문제 수 | 7문제 | 2문제 | 4~5문제 |
| 전략 | 쉬운 문제 먼저 | 정확성 집중 | 빠르게 많이 |

### 문제 풀이 전략

```
[시작 5분]
1. 모든 문제를 훑어본다
2. 난이도를 파악하고 풀 순서를 정한다
   → 확실히 풀 수 있는 문제부터

[문제당 시간 배분]
1. 읽기 + 이해: 5분
2. 알고리즘 선택: 3분
3. 구현: 20~40분
4. 테스트 + 디버깅: 5~10분

[남은 시간]
- 못 푼 문제 부분 점수 확보
- 풀었던 문제 엣지 케이스 재확인
```

### 유형별 접근 체크리스트

| 유형 | 첫 번째 질문 | 핵심 행동 |
|------|-------------|----------|
| 구현 | 조건이 몇 개인가? | 체크리스트 작성 |
| 문자열 | 파싱 규칙은? | split/substring 계획 |
| 그래프 | 최단? 전체 탐색? | BFS vs DFS 선택 |
| DP | dp[i]의 정의는? | 점화식 세우기 |
| 브루트포스 | N의 크기는? | 시간 내 가능한지 계산 |

### 디버깅 전략

1. **예제가 안 맞을 때**: 손으로 트레이싱. 중간값 출력
2. **예제는 맞는데 제출하면 틀림**: 엣지 케이스 확인 (0, 1, 최대값, 빈 입력)
3. **시간 초과**: 알고리즘 자체를 교체. 상수 최적화로는 해결 안 됨

**문제 C**: "주어진 단어 목록에서 애너그램 그룹을 묶어라."
- 답: 해시 (정렬된 문자열을 key로 HashMap 그룹핑)

## 5. 함정 노트

### 1. 시간 배분 실패

어려운 문제에 집착하여 쉬운 문제를 놓치는 것이 최악의 시나리오. **확실한 점수부터 확보.**

### 2. 입출력 형식 실수

```java
// 틀림: 마지막에 공백이나 줄바꿈이 추가됨
for (int i = 0; i < n; i++) {
    System.out.println(arr[i] + " ");  // 뒤에 공백
}

// 맞음: StringBuilder로 정확한 형식
StringBuilder sb = new StringBuilder();
for (int i = 0; i < n; i++) {
    if (i > 0) sb.append(" ");
    sb.append(arr[i]);
}
System.out.println(sb);
```

### 3. 제출 전 클래스명 확인

- 백준: `Main`
- 프로그래머스: `Solution`
- SWEA: `Solution`

클래스명이 틀리면 런타임 에러.

### 4. Scanner vs BufferedReader

N이 10만 이상이면 Scanner는 느릴 수 있다. BufferedReader + StringTokenizer를 사용.

```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
StringTokenizer st = new StringTokenizer(br.readLine());
int n = Integer.parseInt(st.nextToken());
```

### 5. 디버깅 출력 제거 안 함

```java
System.out.println("DEBUG: " + value);  // 제출 전에 반드시 제거!
```

### 문제 1: [백준 1260 - DFS와 BFS](https://www.acmicpc.net/problem/1260)

- **난이도**: Silver III
- **핵심**: 1차원 DP 복습
- **힌트**: dp[i] = max(dp[i-2]+stair[i], dp[i-3]+stair[i-1]+stair[i])

## 7. 보스 문제

### 종합 실전 모의고사

**시간 제한 2시간**으로 아래 2문제를 풀어라.

### [백준 14502 - 연구소](https://www.acmicpc.net/problem/14502)

- **난이도**: Gold IV
- **핵심**: 브루트포스(벽 3개 배치) + BFS(바이러스 확산)
- **포인트**: 빈 칸 중 3개를 조합으로 선택 → 벽 세움 → BFS로 바이러스 확산 → 안전 영역 크기 계산 → 최대값 갱신
- **힌트**: 조합(nC3) + BFS. 배열 복사 필수

### [백준 12865 - 평범한 배낭](https://www.acmicpc.net/problem/12865)

- **난이도**: Gold V
- **핵심**: 0/1 Knapsack DP
- **포인트**: 2차원 DP 테이블. dp[i][j] = i번째 물건까지 고려, 용량 j일 때 최대 가치
- **힌트**: week5에서 배운 배낭 점화식 그대로 적용
