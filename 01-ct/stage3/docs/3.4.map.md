## 1. 설명

### Map이란

**키(Key) - 값(Value) 쌍**을 저장하는 자료구조. 키는 중복 불가, 값은 중복 가능.

```java
import java.util.HashMap;
import java.util.Map;

Map<String, Integer> map = new HashMap<>();
map.put("apple", 3);
map.put("banana", 5);
```

### HashMap

해시 테이블 기반. 키로 O(1) 접근.

### 주요 메서드

| 메서드 | 동작 | 시간 복잡도 |
|--------|------|:-----------:|
| `put(k, v)` | 키-값 추가 (키 이미 있으면 값 덮어쓰기) | O(1) |
| `get(k)` | 키로 값 조회 (없으면 null) | O(1) |
| `getOrDefault(k, d)` | 키로 조회, 없으면 기본값 d 반환 | O(1) |
| `containsKey(k)` | 키 존재 여부 | O(1) |
| `containsValue(v)` | 값 존재 여부 | O(n) |
| `remove(k)` | 키-값 삭제 | O(1) |
| `size()` | 쌍의 수 | O(1) |
| `keySet()` | 모든 키를 Set으로 반환 | O(1) |
| `values()` | 모든 값을 Collection으로 반환 | O(1) |
| `entrySet()` | 모든 키-값 쌍을 Set으로 반환 | O(1) |

### Map 순회

```java
// keySet 순회
for (String key : map.keySet()) {
    System.out.println(key + ": " + map.get(key));
}

// entrySet 순회 (더 효율적)
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}
```

### 자주 쓰는 패턴: 빈도수 세기

```java
int[] arr = {1, 2, 2, 3, 3, 3};
Map<Integer, Integer> freq = new HashMap<>();

for (int num : arr) {
    freq.put(num, freq.getOrDefault(num, 0) + 1);
}
// {1=1, 2=2, 3=3}
```

`getOrDefault`는 코테에서 가장 자주 쓰는 Map 메서드다.

### 자주 쓰는 패턴: 그룹핑

```java
String[] words = {"apple", "ant", "banana", "avocado", "blueberry"};
Map<Character, List<String>> groups = new HashMap<>();

for (String w : words) {
    char first = w.charAt(0);
    if (!groups.containsKey(first)) {
        groups.put(first, new ArrayList<>());
    }
    groups.get(first).add(w);
}
// {a=[apple, ant, avocado], b=[banana, blueberry]}
```

## 2. 예제

### 예제 1: 기본 사용

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Map<String, Integer> scores = new HashMap<>();
        scores.put("Alice", 95);
        scores.put("Bob", 87);
        scores.put("Charlie", 92);

        System.out.println(scores.get("Bob"));         // 87
        System.out.println(scores.containsKey("Dave")); // false
        System.out.println(scores.getOrDefault("Dave", 0)); // 0

        // 값 수정
        scores.put("Bob", 90);  // 기존 키 → 값 덮어쓰기
        System.out.println(scores.get("Bob"));  // 90
    }
}
```

### 예제 2: 빈도수 세기

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        String text = "hello world hello java hello";
        String[] words = text.split(" ");

        Map<String, Integer> freq = new HashMap<>();
        for (String w : words) {
            freq.put(w, freq.getOrDefault(w, 0) + 1);
        }

        // 출력
        for (Map.Entry<String, Integer> e : freq.entrySet()) {
            System.out.println(e.getKey() + ": " + e.getValue());
        }
        // hello: 3, world: 1, java: 1
    }
}
```

### 예제 3: Map + 정렬

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();
        map.put("banana", 3);
        map.put("apple", 5);
        map.put("cherry", 1);

        // 키 기준 정렬
        List<String> keys = new ArrayList<>(map.keySet());
        Collections.sort(keys);

        for (String key : keys) {
            System.out.println(key + ": " + map.get(key));
        }
        // apple: 5, banana: 3, cherry: 1
    }
}
```

## 4. 이론 퀴즈

**Q1.** `put()`으로 이미 존재하는 키에 새 값을 넣으면?

<details>
<summary>정답</summary>
기존 값이 새 값으로 덮어써진다. 이전 값이 반환된다.
</details>

**Q2.** `getOrDefault("key", 0)`의 동작은?

<details>
<summary>정답</summary>
"key"가 존재하면 해당 값을 반환하고, 없으면 기본값 0을 반환한다.
</details>

**Q3.** `keySet()`과 `entrySet()`의 차이는?

<details>
<summary>정답</summary>
keySet()은 키만 Set으로 반환. entrySet()은 키-값 쌍(Entry)을 Set으로 반환. entrySet이 순회 시 더 효율적이다 (get() 호출 불필요).
</details>

## 5. 함정 노트

### 1. get()에서 null 반환

```java
Map<String, Integer> map = new HashMap<>();
int value = map.get("unknown");  // NullPointerException!
// Integer 언박싱 시 null → int 변환 불가

// 해결
int value = map.getOrDefault("unknown", 0);
```

### 2. put()은 기존 값을 덮어쓴다

```java
map.put("key", 1);
map.put("key", 2);  // 값이 1 → 2로 변경
```

누적하려면 `getOrDefault` 사용:
```java
map.put("key", map.getOrDefault("key", 0) + 1);
```

### 3. HashMap의 순서

```java
Map<String, Integer> map = new HashMap<>();
// 순회 순서 ≠ 삽입 순서!

// 삽입 순서가 필요하면
Map<String, Integer> ordered = new LinkedHashMap<>();
```

### 4. containsKey vs containsValue 성능 차이

```java
map.containsKey("key");    // O(1) — 해시 조회
map.containsValue(100);    // O(n) — 모든 값을 순회
```
