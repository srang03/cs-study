## 스킵 클리어 기록

이론 퀴즈 전부 정답 시 직접 작성을 스킵한 단원 목록:

(해당 단원의 직접 작성 항목을 `- [-] 직접 작성 (스킵)`으로 변경)

---

## 사용법

1. 각 항목 완료 시 `[ ]` → `[x]`로 변경
2. 이론 퀴즈: `_` 자리에 `O` 또는 `X` 기록 (예: `Q1:O Q2:X Q3:O`)
3. 스킵 클리어: 이론 퀴즈 전부 O일 때 직접 작성을 `[-] 직접 작성 (스킵)`으로 변경
4. 랭크 달성 시 상단 "현재 랭크" 업데이트
5. 보스 회고는 해당 단원 `practice/review.md`에 작성 (`stage1/practice/review_template.md` 복사)
