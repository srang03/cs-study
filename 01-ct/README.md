# 코딩 테스트 준비 (01-ct)

자바 기초 문법 및 코딩 학습 커리큘럼. 프로그래머스·백준 코딩 테스트 준비를 위한 실습 중심 학습.

## 학습 순서

```
stage1 → prep → stage2 → stage3 → stage4
```

- **stage1**: 기초 문법 (환경, 입출력, 변수·타입, 연산자, 메서드)
- **prep**: 알고리즘 준비 (제어문, 배열, 예외·StringBuilder)
- **stage2**: OOP (클래스, 상속·다형성, 인터페이스)
- **stage3**: 컬렉션 (제너릭, List, Set, Map)
- **stage4**: 알고리즘 (5주)
